Global Inequality Trend Analysis and Visualization

Dataset: Global Consumption and Income Project (GCIP): 70,000 observations; 142 countries; 35 years.

Goal: Quantify how top and bottom income inequality affect systemic economic growth and provide automated, reproducible country insights.

What I did:

)Data cleaning, structuring and preparation.

)Automated single-country and cross‑country trend visualization analysis.

)Built an interactive Tableau dashboard.

Key findings:

Bottom inequality: significant negative effect on growth.

Top inequality: neutral to positive effect.

Tech stack  
Excel; R (RStudio); Tableau
